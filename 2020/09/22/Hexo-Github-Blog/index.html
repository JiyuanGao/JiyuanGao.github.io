Hexo+Github=Blog

2020-09-22 | Tech

node git hexo github

Before

Install node

From official website, we can install the package.

Install git

From official website, we can install the package.

Install hexo

mkdir myblog
cd myblog
npm install hexo-cli -g
hexo init

Steps

Create repository on github

Please follow instructions to create a new repository.

Clone repository

git clone git@github.com:USERNAME/USERNAME.github.io.git

Generate and server

hexo generate

hexo server

Configure __config.yml

Install hexo-deployer-git

npm install hexo-deployer-git -save

Deploy

hexo deploy

End

https://USERNAME.github.io

Change the theme?

Change your blog’s theme: https://hexo.io/themes/

Download it and put it in theme directory

Change the directory’s name: hexo-theme-next->next

In config.yml: theme: next

New post?

hexo new "NEW POST"

edit NEW POST.md

hexo g -d
